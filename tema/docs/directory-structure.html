Dashy

Files

Dashy Laravel 5 consists of Laravel 5 blade template files.
  • Base layout files are located in resources/views/layouts/
  • The blade template pages are located in resources/views/
  • The widgets are located in resources/views/widgets/
  • The default bootstrap less files are located in resources/assets/less/boostrap/
  • There are specific less files for separate widgets and pages which are located in resources/assets/less/widgets and resources/assets/less/pages respectively.

Theme Colors

Currently there are 3 different theme colors for Dashy Laravel 5 theme, other than the default one. You can choose each one from the Color Select button on right side of the navigation bar. To select a theme color, you have to include the theme specific .css file in your plain.blade.php. The available theme colors are

  • app-green.css (Default)
  • app-red.css
  • app-blue.css
  • app-orange.css
